# RareBook Archive

A blockchain-based platform for authenticating and managing rare book collections, providing provenance tracking and literary heritage preservation.

## Features

- Rare book authentication and cataloging
- Publication year validation
- Genre classification system
- Rarity assessment tracking
- Loan management system

## Smart Contract Functions

### Public Functions
- `catalog-rare-book`: Register a rare book in archive
- `loan-book`: Update book loan status

### Read-Only Functions
- `get-book-record`: Retrieve book information
- `get-librarian`: Get librarian's address

## Supported Genres
- Literature, History, Philosophy, Science, Poetry, Biography

## Rarity Classifications
- Extremely Rare, Very Rare, Rare, Uncommon, Scarce
